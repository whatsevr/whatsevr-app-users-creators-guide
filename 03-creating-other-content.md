# Creating Other Content

## Introduction

Whatsevr offers multiple ways to express yourself and share content with your audience. While WTV videos and Flicks dominate for video content, there are several other powerful content types to showcase your creativity, promote opportunities, and build your professional presence.

### Content Types Overview

- **Photo Posts**: Share multiple images with captions, location tags, and user mentions to your personal profile, communities, or portfolio
- **Memories**: Temporary story-like content (images or videos) that stays visible for 1-3 days before disappearing
- **Offers**: Promotional content for job postings, collaborations, services, and deals with rich text formatting
- **PDF Documents**: Upload professional documents, portfolios, resumes, and certificates to your portfolio

Each content type serves different purposes and supports different features. Choose the one that best fits your content and goals.

## Section 1: Creating Photo Posts

### What Are Photo Posts?

Photo posts allow you to share up to 10 images at once with captions, location information, and user/community tags. Perfect for:
- Sharing multiple moments from an event
- Showcasing projects or work samples (portfolio)
- Telling stories through image sequences
- Sharing collections of related content

### Step-by-Step Guide to Creating Photo Posts

#### Step 1: Access the Create Photo Post Page

1. Tap the **Create** button (+ icon) on your home feed or profile
2. Select **Photo Post** from the content type menu
3. You'll be taken to the photo post creation page

[Screenshot: Photo post creation page with empty state and "Add Images" button]

#### Step 2: Add Images

1. Tap **Add Images** or the media selection area
2. Your device's gallery will open
3. Select up to 10 images by tapping each one
4. Selected images will show a checkmark
5. Tap **Done** or **Confirm** to add them to your post

**Important**: You can add images in multiple stages:
- Add some images initially
- After reviewing, you can add more images using the **Add More** button
- Remove unwanted images by tapping the delete (X) icon on each image

[Screenshot: Gallery picker showing image selection with checkmarks]

[Screenshot: Photo post with multiple images in preview grid]

#### Step 3: Preview Your Images

- Your images appear in a swipeable preview grid
- Swipe left/right to view each image
- Long-press on any image for options to:
  - View full preview
  - Rotate the image
  - Remove the image
  - Edit the image

[Screenshot: Swipeable image preview in photo post creation]

#### Step 4: Edit Individual Images

Each image can be edited using the built-in image editor:

1. Tap on any image in the preview to open the editor
2. Select your desired **Aspect Ratio**:
   - **Square (1:1)**: Balanced and compact
   - **Portrait (4:5)**: Mobile-friendly vertical format
   - **Portrait (9:16)**: Full-screen vertical format
   - **Landscape (3:2)**: Classic wide format
   - **Landscape (16:9)**: Widescreen format
   - **Landscape (21:9)**: Ultra-wide format
   - **Landscape (1.91:1)**: Instagram-style landscape

3. Use the cropping tools:
   - **Pinch to zoom**: Resize the image within the frame
   - **Drag**: Reposition the image
   - **Rotate**: Rotate the image as needed

4. Apply additional editing (if available):
   - Filters and effects
   - Brightness/contrast adjustments
   - Saturation and color adjustments

5. Tap **Done** to save the edited image

[Screenshot: Image cropper with aspect ratio options]

[Screenshot: Image editor with filters and adjustment tools]

#### Step 5: Add Caption and Details

**For Personal/Community Posts:**
- Tap the **Caption** field
- Type your caption (up to 10,000 characters)
- You can use hashtags (#) and mentions (@) for better discoverability

**For Portfolio Posts:**
- **Title** field (required, up to 150 characters):
  - Keep it concise and descriptive
  - Examples: "Summer Design Project", "Event Photography"
  
- **Description** field (optional, up to 2,500 characters):
  - Provide detailed context about your photos
  - Explain the project, location, or story
  - Share your creative process or insights

[Screenshot: Caption field with keyboard visible]

[Screenshot: Portfolio post with Title and Description fields]

#### Step 6: Add Location Tag

1. Tap the **Location** field
2. A location search interface will appear
3. Type the location name or address:
   - Manual search: Type to find a specific place
   - Nearby suggestions: View suggested locations near you
4. Select the location from the list
5. To change location: Tap the X to clear and search again
6. To remove location: Leave it blank

**Tip**: Adding location helps your content appear in location-based searches and increases local visibility.

[Screenshot: Location search interface with suggestions]

#### Step 7: Tag Users and Communities

1. Tap the **Tag Users/Communities** button or field
2. A tag dialog will open
3. Search for users or communities by name:
   - Type the username or community name
   - Results appear as you type
4. Tap to select each person or community you want to tag
5. Selected tags appear in a list below the search field
6. To remove a tag: Tap the X next to the tag name
7. Tap **Done** when finished

**Benefits of tagging**:
- Notifies tagged users/communities
- Increases content visibility
- Encourages engagement and sharing

[Screenshot: Tag users and communities dialog]

[Screenshot: Selected tags displayed below search field]

#### Step 8: Choose Posting Context

Select where your content will be posted:

1. **Personal Profile** (Account):
   - Content appears on your profile
   - Visible to your followers and public
   - Best for personal content and memories

2. **Community**:
   - Content appears in the community feed
   - Visible to community members
   - Requires selecting a specific community
   - Best for content related to community interests

3. **Portfolio** (Professional content):
   - Content appears in your portfolio section
   - Professional presentation with title and description
   - Best for work samples, projects, and professional content
   - Unlocks additional fields (Title, Description)

Select the appropriate context for your post.

[Screenshot: Posting context selection with radio buttons or tabs]

#### Step 9: Submit Your Photo Post

1. Review all your content:
   - Images are in the desired order
   - Caption/Title is complete and descriptive
   - Location is accurate (if added)
   - Tags are correct
   - Posting context is appropriate

2. Tap the **Done** or **Post** button in the top-right or bottom of the screen

3. Wait for the post to upload (progress indicator may appear)

4. On success:
   - You'll see a confirmation message
   - You'll be redirected to the post or your profile
   - Your photo post is now live

[Screenshot: Photo post ready to submit with all details filled]

[Screenshot: Success message "Post published successfully"]

## Section 2: Creating Memories

### What Are Memories?

Memories are temporary, story-like content that appear on your profile for a limited time (1-3 days). Perfect for:
- Sharing fleeting moments
- Ephemeral updates
- Time-sensitive announcements
- Daily stories

After the selected duration, memories automatically disappear from your profile.

### Step-by-Step Guide to Creating Memories

#### Step 1: Access the Create Memory Page

1. Tap the **Create** button (+ icon)
2. Select **Memory** from the content type menu
3. You'll be taken to the memory creation page

[Screenshot: Memory creation page with empty state]

#### Step 2: Choose Between Image or Video

Memories require exactly one media item:

1. Tap **Select Image** or **Select Video** button
2. Your gallery will open
3. Select one image or video:
   - **Image**: Any JPEG, PNG, or WebP format
   - **Video**: MP4 or MOV format
   - Note: You cannot add both image and video to a single memory

[Screenshot: Gallery picker for image selection]

#### Step 3a: If You Selected an Image

1. The image appears in a preview area
2. Default aspect ratio is **Portrait (4:5)**, optimized for mobile viewing
3. Your image is ready for the next step

**To change the image:**
- Tap **Change Image**
- Select a different image from your gallery

[Screenshot: Image preview in memory creation]

#### Step 3b: If You Selected a Video

1. The video appears with a preview thumbnail
2. A play button overlays the preview
3. You can preview the video by tapping the play button

**Select a Custom Thumbnail:**
1. Tap **Select Thumbnail** button
2. Your gallery opens
3. Choose an image to serve as the thumbnail
4. The selected image becomes the preview for your video

**To change the video:**
- Tap **Change Video**
- Select a different video from your gallery

**To update the thumbnail:**
- Tap **Change Thumbnail** or **Update Thumbnail**
- Select a different image

[Screenshot: Video preview with thumbnail selection]

[Screenshot: Custom thumbnail applied to video memory]

#### Step 4: Add Caption

1. Tap the **Caption** field
2. Type your caption (up to 100 characters)
3. You have limited space—be concise and impactful
4. Use this space for a quick message or context about your memory

**Tips for captions:**
- Be direct and engaging
- Use emojis for visual appeal
- Mention key information in the first few words
- Consider your audience before posting

[Screenshot: Caption field with character limit indicator]

#### Step 5: Expand More Details (Optional)

Tap **More Details** to access additional options:

##### 5a: Mention Users and Communities

1. Tap **Add Mentions** or the mentions field
2. Search for users or communities to mention
3. Mentions notify the tagged users/communities
4. Selected mentions appear in a list

##### 5b: Add Location

1. Tap the **Location** field
2. Search for and select a location
3. Location helps with discoverability
4. You can change it anytime before posting

##### 5c: Add User Action (CTA)

1. Tap **User Action** or **Call to Action**
2. Select an action from available options:
   - Visit Website
   - Call Now
   - Message
   - Book Now
   - Learn More
   - And other CTA types

3. **Add Action URL**:
   - Tap the URL field
   - Enter the relevant URL or contact information
   - This is what happens when users tap the CTA button

4. To remove CTA:
   - Tap the X or remove button
   - CTA will be cleared

[Screenshot: More Details expanded showing all optional fields]

[Screenshot: User Action selection with dropdown menu]

#### Step 6: Set Duration (How Long It Stays Up)

1. Tap the **Duration** field
2. Select how long your memory will remain visible:
   - **1 Day**: Memory disappears after 24 hours
   - **2 Days**: Memory disappears after 48 hours
   - **3 Days**: Memory disappears after 72 hours

3. Default is typically 3 days

**Choose based on:**
- How urgent your message is
- How long you want it visible to your audience
- The nature of the temporary content

[Screenshot: Duration selection with day options]

#### Step 7: Choose Posting Context

Select where your memory will be posted:

1. **Personal Profile** (Account):
   - Memory appears on your profile
   - Visible to followers and public
   - Most common choice

2. **Community**:
   - Memory appears in community feed
   - Visible to community members
   - Requires community selection

Select the appropriate context.

[Screenshot: Posting context selection]

#### Step 8: Submit Your Memory

1. Review all details:
   - Image/Video is correct
   - Caption is engaging (100 chars)
   - Duration is appropriate
   - CTA (if added) has valid URL
   - Location is accurate (if added)

2. Tap **Done** or **Post** button

3. Wait for upload completion

4. You'll see a confirmation and be redirected

[Screenshot: Memory ready to submit]

[Screenshot: Memory posted successfully notification]

### Best Practices for Memories

- **Timing**: Post memories when your audience is most active
- **Urgency**: Use shorter durations (1-2 days) for time-sensitive content
- **Engagement**: Use CTA actions to drive interactions
- **Mentions**: Tag relevant users to increase reach and notifications
- **Variety**: Mix different types of memories (photos, videos, CTAs)
- **Consistency**: Post memories regularly to keep your audience engaged

## Section 3: Creating Offers

### What Are Offers?

Offers are promotional content for:
- Job postings and career opportunities
- Freelance collaborations and partnerships
- Service promotions and deals
- Business opportunities and ventures

Offers support rich text formatting, multiple media items, and detailed targeting options.

### Step-by-Step Guide to Creating Offers

#### Step 1: Access the Create Offer Page

1. Tap the **Create** button (+ icon)
2. Select **Offer** from the content type menu
3. You'll be taken to the offer creation page

[Screenshot: Offer creation page with empty form]

#### Step 2: Add Title

1. Tap the **Title** field
2. Type your offer title (up to 150 characters)
3. Make it clear and compelling

**Title examples:**
- "Hiring: Senior Designer - Remote"
- "Collaboration Wanted: Content Creator"
- "Service: Professional Logo Design"
- "Deal: 50% Off Web Development"

**Tips for titles:**
- Start with the main action or category
- Include key details (location, type, level)
- Make it searchable with relevant keywords

[Screenshot: Title field in offer creation]

#### Step 3: Add Media (Images and Videos)

Offers support up to **5 images + 2 videos** (7 total media items):

1. Tap **Add Media**, **Select Files**, or the media area
2. Your gallery opens
3. Select images and/or videos:
   - Select up to 5 images (JPEG, PNG, WebP)
   - Select up to 2 videos (MP4, MOV)
   - Mixed media is supported

4. Tap **Done** to add them

**Managing media:**
- Preview media in a swipeable grid
- Swipe left/right to view each item
- Tap the delete (X) icon to remove a media item
- Add more items using **Add More Media** button

**For video media:**
- A thumbnail will be automatically generated or you can select a custom thumbnail
- Tap on a video to select/change its thumbnail

[Screenshot: Media selection with mixed images and videos]

[Screenshot: Offer with multiple media items in preview]

#### Step 4: Add Description

Choose between plain text or rich text formatting:

##### Option A: Plain Text Description

1. Tap the **Description** field
2. Type your description (up to 10,000 characters)
3. Simple text format, no styling
4. Good for quick, straightforward offers

##### Option B: Rich Text Description (Description Designer)

For a more professional look with formatting:

1. Tap **Description Designer** or formatting option
2. A rich text editor will open (delta editor)
3. Available formatting options:
   - **Bold**: Make text bold for emphasis
   - **Italic**: Italicize text
   - **Underline**: Underline important text
   - **Lists**: Create bullet points or numbered lists
   - **Headings**: Use heading styles for structure
   - **Color**: Change text color
   - **Alignment**: Left, center, or right alignment

4. Type and format your description:
   - Add section headings for clarity
   - Use bullet points for key benefits or requirements
   - Bold important terms or deadlines
   - Organize information logically

5. Tap **Save** when finished formatting

**To edit or clear formatted description:**
- Tap **Edit Description** to modify
- Tap **Clear Description** to remove formatting and start fresh

[Screenshot: Description Designer with formatting toolbar]

[Screenshot: Formatted description with headings, bold text, and bullet points]

**Tips for descriptions:**
- Be detailed but concise
- Highlight key requirements or benefits
- Use formatting for readability
- Include deadlines or important dates
- Provide next steps or how to respond

#### Step 5: Add Status (Required)

1. Tap the **Status** field
2. Select from predefined statuses:
   - **Hiring**: Job opening or recruitment
   - **Searching**: Looking for something specific
   - **Collaborating**: Open to partnerships
   - **Closed**: Offer no longer available
   - Other status options may be available

3. Or enter a **Custom Status**:
   - Tap to type a custom status if needed
   - Examples: "Limited Time", "Urgent", "Featured"

4. Status helps users understand the nature of your offer

[Screenshot: Status selection dropdown]

#### Step 6: Add Target Area (Optional but Recommended)

Define where your offer applies:

1. Tap **Add Target Area** button
2. Select geographic scope:
   - **Country**: Select target country
   - **State/Region**: Select specific state (if applicable)
   - **City**: Select specific city for local offers

3. You can add multiple target areas:
   - Tap **Add Another Area** to add more
   - Useful for offers available in multiple locations

4. To remove a target area:
   - Tap the X next to the area name

**Examples:**
- Global offers: Select only country
- Regional offers: Select country + state
- Local offers: Select country + state + city

[Screenshot: Target area selection with country, state, city dropdowns]

#### Step 7: Expand More Details (Optional)

Tap **More Details** to access:

##### 7a: Set Target Audience/Gender

1. Tap **Audience** or **Target Gender**
2. Select from options:
   - **All**: Open to everyone
   - **Men**: Target male audience
   - **Women**: Target female audience

3. This helps filter the offer to relevant audiences

##### 7b: Add User Action (CTA)

1. Tap **User Action** or **Call to Action**
2. Select an action:
   - **Apply Now**: For job postings
   - **Visit Website**: To external link
   - **Message Now**: Direct messaging
   - **Book Now**: For services
   - **Learn More**: For additional information
   - Other CTA options

3. Add **Action URL**:
   - Required for most CTAs
   - Include full URL for websites
   - Phone number for calls
   - Email for inquiries

[Screenshot: User Action selection with CTA options]

##### 7c: Tag Users and Communities

1. Tap **Add Tags** or tag field
2. Search for users or communities
3. Select relevant tags
4. Tagged entities get notified
5. Increases visibility in their feeds

[Screenshot: Tag selection for offer]

#### Step 8: Choose Posting Context

Select where your offer will be posted:

1. **Personal Profile** (Account):
   - Appears on your profile
   - Visible to followers and public

2. **Community**:
   - Appears in community feed
   - Visible to community members
   - Requires community selection

**Tip**: Community posts reach engaged members interested in that topic.

#### Step 9: Review and Submit Offer

1. Check all details:
   - Title is clear and compelling
   - Media (images/videos) are relevant
   - Description is complete and well-formatted
   - Status accurately reflects offer state
   - Target area is set correctly (if needed)
   - CTA has valid URL/contact info
   - Audience selection is appropriate

2. Tap **Done** or **Post** button

3. Wait for offer to be published

4. Confirmation message appears
5. Navigate to view your published offer

[Screenshot: Offer ready to submit with all details]

[Screenshot: Offer posted successfully]

### Best Practices for Offers

- **Title**: Make it specific and searchable
- **Description**: Use Description Designer for professional formatting
- **Media**: Include high-quality images or videos showcasing your offer
- **Status**: Keep status updated (mark as "Closed" when no longer available)
- **Targeting**: Set accurate geographic and audience targeting
- **CTA**: Always include a clear call-to-action with valid URL/contact
- **Details**: Be comprehensive but organized
- **Frequency**: Post offers regularly, but avoid spam

## Section 4: Uploading PDF Documents

### What Are PDF Uploads?

PDF uploads allow you to share professional documents in your portfolio:
- Resumes and CVs
- Portfolio pieces and case studies
- Certificates and credentials
- Brochures and flyers
- Business documents

**Note**: PDF uploads are portfolio-only and require portfolio access.

### Step-by-Step Guide to Uploading PDFs

#### Step 1: Access Upload PDF Page

1. Tap the **Create** button (+ icon)
2. Select **Upload PDF** or **PDF Document**
3. You'll be taken to the PDF upload page

[Screenshot: PDF upload page with empty state]

#### Step 2: Select PDF File

1. Tap **Select PDF** or **Choose File**
2. Your device's file browser opens
3. Navigate to your PDF file location
4. Select the PDF file you want to upload
5. File size will be displayed (max 1 GB)

**Supported formats:**
- PDF (.pdf) only

**Tips:**
- Ensure your PDF is not corrupted
- Check file size before selecting
- Have your best PDFs organized in a folder

[Screenshot: File picker showing PDF file selection]

## Section 6: Understanding Posting Contexts

Every content type supports different posting contexts. Choose the right one for your content:

### Personal Profile (Account)

- **What**: Content appears on your personal profile
- **Visibility**: Visible to your followers and public (depending on privacy settings)
- **Use for**: Personal moments, casual content, everyday sharing
- **Example**: Photos from a weekend trip, daily memories

### Community Posts

- **What**: Content appears in the community feed
- **Visibility**: Visible to community members
- **Requirements**: Must select a specific community during creation
- **Use for**: Content relevant to community interests and topics
- **Example**: Design portfolio photo to a Design Community, job offer to a Hiring Community

**Benefits:**
- Reaches engaged community members
- Increases visibility within specific groups
- Encourages targeted engagement

### Portfolio (Professional Showcase)

- **What**: Content appears in your portfolio section
- **Visibility**: Professional presentation, prominently displayed
- **Requirements**: Portfolio access enabled in your account
- **Additional Fields**: Title and description fields for more details
- **Use for**: Professional work, projects, credentials, and services
- **Example**: Professional photography portfolio, design projects, certificates

**Benefits:**
- Professional presentation
- Detailed descriptions for context
- Ideal for showcasing expertise
- Attracts potential clients or employers

3. Edited image is saved to your post
4. The cropping and editing are finalized
5. Image is ready for caption and other details

**Note**: Edits are applied to each individual image. If you want different crops for different images, edit each one separately.

## Section 8: Technical Requirements & Limits

### File Size Limits

- **Maximum upload size**: 1 GB per individual file
- **Recommended sizes**:
  - Images: 500 KB - 50 MB for optimal performance
  - Videos: 100 MB - 500 MB (depending on resolution)

- **Supported formats**: JPEG, PNG, WebP
- **Recommended resolution**:
  - Minimum: 600x600 pixels
  - Optimal: 1080x1080 pixels or higher
  - Maximum: 8000x8000 pixels

### Video Requirements

For Memories and Offers:

- **Supported formats**: MP4, MOV
- **Recommended duration**:

  - Offers: 5 seconds - 5 minutes
  
- **Resolution**: 720p (1280x720) minimum, 4K (3840x2160) supported

- **Frame rate**: 24 fps - 60 fps recommended

### PDF Requirements

- **Supported format**: PDF (.pdf) only
- **Maximum file size**: 1 GB
- **Recommended size**: 5 MB - 100 MB
- **Pages**: 1 - 1000 pages (limit may apply)

## Section 10: Troubleshooting Common Issues

### Cannot Add More Images

**Problem**: You've reached the image limit but want to add more.

**Solutions**:
- Check if you've hit the limit (10 for photos, 5 for offers)
- Remove unwanted images first using the delete (X) button
- Tap **Add More** to add additional images after review
- Start with a smaller selection and add more later

### Image Upload Fails

**Problem**: Images won't upload or selection fails.

**Solutions**:
- **Check file size**: Ensure individual image is under 1 GB
- **Verify format**: Use JPEG, PNG, or WebP only
- **Check connection**: Verify stable internet connection
- **Clear cache**: Go to app settings and clear app cache
- **Restart app**: Close and reopen the Whatsevr app
- **Try different image**: Select a different image to isolate the issue
- **Update app**: Ensure you have the latest app version

### Caption or Description Too Long

**Problem**: Character limit exceeded message appears.

**Solutions**:
- **Check limits**: Review character limits for your content type
  - Photo caption: 10,000 (personal/community) or 150 (portfolio title)
  - Memory caption: 100
  - Offer title: 150, description: 10,000
  - PDF title: 100, description: 10,000
  
- **Edit text**: Remove unnecessary text to fit within limit
- **Use Description Designer**: For offers, use the rich text editor instead of plain text
- **Be concise**: Make your message more direct and concise
- **Break into parts**: Consider splitting content across multiple posts

### Location Not Found

**Problem**: Cannot find desired location in search.

**Solutions**:
- **Check spelling**: Verify location name spelling
- **Try variations**: Use different keywords (city name, landmarks, etc.)
- **Check connection**: Ensure internet connection is stable
- **Use nearby suggestions**: Select from suggested nearby locations
- **Manually type**: Type the location name if it's not in suggestions
- **Update location data**: Restart app to refresh location database

### Cannot Tag Users or Communities

**Problem**: Tag search returns no results or tags won't appear.

**Solutions**:
- **Verify internet**: Check stable internet connection
- **Check username**: Ensure exact username spelling
- **Account exists**: Verify user or community actually exists
- **Privacy settings**: User account might have disabled tagging
- **Try shorter name**: Search with fewer characters
- **Wait for results**: Sometimes search takes a moment to load
- **Restart app**: Close and reopen Whatsevr

### PDF Upload Fails

**Problem**: PDF file won't upload.

**Solutions**:
- **Verify format**: Ensure file is actual PDF (.pdf extension)
- **Check file size**: PDF must be under 1 GB
- **Corrupt file**: Try a different PDF
- **Internet connection**: Ensure stable connection
- **Compress PDF**: If large, try compressing before upload
- **Try later**: Retry after some time
- **Restart**: Close app and try again

### Thumbnail Not Showing

**Problem**: Thumbnail doesn't appear or shows blank.

**Solutions**:
- **Re-select thumbnail**: Tap "Change Thumbnail" and select again
- **Check image format**: Use JPEG or PNG format
- **Try different image**: Select a different image as thumbnail
- **Verify image**: Ensure image opens properly
- **Clear cache**: Clear app cache in settings
- **Restart app**: Close and reopen Whatsevr

### Description Designer Not Saving

**Problem**: Formatted description doesn't save or gets lost.

**Solutions**:
- **Check connection**: Ensure stable internet connection
- **Content not empty**: Ensure there's text in the description
- **Tap Save**: Explicitly tap "Save" or "Done" button
- **Wait for save**: Sometimes formatting takes time to process
- **Try plain text**: If issues persist, use plain text description instead
- **Restart app**: Close and reopen Whatsevr
- **Report issue**: If persistent, report to support

### Post Submission Fails

**Problem**: "Post failed" or "Upload error" message appears.

**Solutions**:
- **Check internet**: Verify stable connection (WiFi or mobile data)
- **Fill required fields**: Ensure all required fields are completed
  - Photo post: Images and caption (if applicable)
  - Memory: Image/video and caption
  - Offer: Title and status
  - PDF: PDF file and title

- **Verify media**: Ensure media files are valid and not corrupted
- **Try again**: Wait a moment and retry submission
- **Reduce file size**: If media is large, try compressing
- **Restart app**: Close and reopen Whatsevr
- **Update app**: Ensure you're running the latest version
- **Contact support**: If error persists, report to support team

### Media Preview Shows Incorrectly

**Problem**: Images/videos appear distorted or cut off in preview.

**Solutions**:
- **Check aspect ratio**: Verify selected aspect ratio matches content
- **Adjust crop**: Re-edit image with different crop positioning
- **Try different ratio**: Select a different aspect ratio option
- **Replace media**: Select a different image or video
- **Clear preview cache**: Restart app to refresh preview
